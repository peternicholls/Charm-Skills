# Code of Conduct

## Our Pledge

We pledge to make participation in this project respectful, welcoming, and productive for everyone.

## Expected Behavior

- Be respectful and constructive.
- Focus criticism on ideas, behavior, and evidence.
- Assume good intent while still asking for clarity.
- Keep discussions relevant to improving the Skills and repository.

## Unacceptable Behavior

- Harassment, threats, or discriminatory language.
- Personal attacks.
- Publishing private information without permission.
- Sustained disruption of project discussions.

## Enforcement

Maintainers may remove, edit, or reject comments, commits, issues, and pull requests that violate this Code of Conduct. Maintainers may also temporarily or permanently ban contributors for behavior they deem inappropriate, threatening, offensive, or harmful.

Report concerns privately to the maintainers once contact details are available in the GitHub repository profile or issue tracker.

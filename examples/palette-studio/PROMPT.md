# Prompt

Build a terminal palette studio that lets designers preview a few terminal UI themes. It should show color swatches, sample buttons, alerts, cards, and a responsive preview. Use Charm Skills so the app showcases Lip Gloss polish, Bubble Tea key handling, and deterministic visual output.

## Expected Skills

- `charm-tui-builder`
- `charm-lipgloss-layout`
- `charm-tui-qa`

## Acceptance Criteria

- Runs locally without network calls.
- Uses Bubble Tea for theme switching.
- Uses Lip Gloss for color, spacing, borders, swatches, and responsive layout.
- Includes multiple named themes.
- Includes deterministic sample output for CI and quick inspection.
- Includes tests for theme cycling and rendered preview content.
